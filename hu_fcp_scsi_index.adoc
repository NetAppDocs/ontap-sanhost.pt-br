= Saiba como configurar hosts SAN do ONTAP com FCP e iSCSI
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Você pode configurar determinados sistemas operacionais (SO) de host SAN para FCP e iSCSI com ONTAP como destino. Se o seu SO de host for compatível com o software SAN Host Utilities, primeiro você instala o pacote de utilitários de host do SO relevante, que inclui o SAN tool kit. Em seguida, para todos os tipos de SO de host, você configura as definições de multipath para LUNs do ONTAP.
