= Saiba mais sobre o suporte e os recursos do ONTAP para hosts Proxmox.
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Os recursos suportados para configuração de host com NVMe over Fabrics (NVMe-oF) variam de acordo com a versão do ONTAP e do Proxmox.

[cols="30,20,20"]
|===
| Recurso ONTAP | versão do host Proxmox | Versão ONTAP 


| Fornece detalhes do ONTAP para namespaces NVMe/FC e NVMe/TCP usando o recurso nativo. `nvme-cli` pacote.  a| 
9,0
| 9.10.1 ou posterior 


| NVMe/TCP é um recurso empresarial totalmente suportado.  a| 
9,0
| 9.10.1 ou posterior 


| O tráfego NVMe e SCSI é suportado no mesmo host usando NVMe multipath para namespaces NVMe-oF e dm-multipath para LUNs SCSI.  a| 
9,0
| 9.4 ou posterior 
|===
O ONTAP suporta os seguintes recursos de host SAN, independentemente da versão do ONTAP em execução na configuração do seu sistema.

[cols="70,30"]
|===
| Recurso | versão do host Proxmox 


| O multipathing NVMe nativo está ativado por padrão.  a| 
9,0



| O `nvme-cli` O pacote inclui scripts de conexão automática, eliminando a necessidade de scripts de terceiros.  a| 
9,0



| A regra udev nativa em `nvme-cli` Este pacote fornece balanceamento de carga round-robin para NVMe multipath.  a| 
9,0

|===

NOTE: Para obter detalhes sobre as configurações suportadas, consulte olink:https://mysupport.netapp.com/matrix/["Ferramenta de Matriz de interoperabilidade"^] .
